# JQuery (https://jquery.com/)

- 자바스크립트를 좀 더 짧고 효율적으로 쓰고자 하는 라이브러리
- document.querySelector('#id') => $('#id')
- button1.addEventListener('click') ==> button1.on('click')
- 자바스크립트를 사용하면 단어가 너무 길어서 불편했던 점들을 개선하기 위한 라이브러리

# JQuery CDN (https://releases.jquery.com/)
 - uncompressed : 보기 좋게 들여쓰기가 잘 되어있는 버전
 - minified : 파일 자체의 용량을 줄이기 위해 변수명 및 들여쓰기가 최소화 된 버전 
	- https://code.jquery.com/jquery-3.7.1.min.js (버전은 업데이트 될 수 있음 / 현재 사용하는 버전임)
 - slim : 꼭 필요한 기능만 넣은 버전
 - slim minified : 꼭 필요한 기능만 포함 + minified
